# Requirements

- Start up with a command
- Observe different kind of tests:
  - Unit test, component tests
  - Integration tests
  - Performance tests
- Telemetry for:
  - Traces
  - Logs
  - Metrics

## Run stack

This stack is optimized to run on unix like system.
At the root folder run follow command to build & start the stack.

```
make build
make up
```
